SHE Key Update Calculator

HIS SHE specification — M1 M2 M3 M4 M5 computation

Inputs

All values in HEX. UID = 30 hex chars (120 bits). Keys = 32 hex chars (128 bits). Counter = 1-268435455 (28-bit decimal).
MASTER_ECU_KEY (AuthKey) — 32 hex chars
New Key to Load — 32 hex chars
ECU UID — 30 hex chars (120 bits)
Counter (decimal, 28-bit max: 268435455)
Key Slot ID (target)
Auth Key Slot (AuthID)
Flags (FID)

Derived Keys

K1 Encryption key (AES-ECB of ENC_C)
K2 MAC key (AES-ECB of MAC_C)

Messages

1
M1 — ECU identity (UID | ID | AuthID) 16 bytes
2
M2 — Encrypted new key (AES-CBC with K1) 32 bytes
3
M3 — Authentication MAC (CMAC of M1|M2 with K2) 16 bytes
4
M4 — Proof of update (AES-ECB with NewKey) 16 bytes
5
M5 — MAC over M4 (CMAC with K2) 16 bytes

CMD_LOAD_KEY payload (M1 + M2 + M3)
